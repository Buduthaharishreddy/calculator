# Pastel Calculator 🎨

A simple, responsive desktop calculator built with Python and Tkinter, featuring a soothing pastel color palette.

## ✨ Features

- **Standard Arithmetic**: Supports addition, subtraction, multiplication, and division.
- **Pastel UI**: Beautiful design using a curated pastel color scheme (`#FDFD96`, `#C7CEEA`, etc.).
- **Responsive Layout**: The window is resizable and adapts to different sizes.
- **Error Handling**: Gracefully handles division by zero and invalid mathematical expressions.
- **Clean Code**: Modular architecture with clear separation between UI (`ui.py`) and calculation logic (`logic.py`).

## 🚀 How to Run

1.  **Requirement**: Ensure you have Python 3 installed.
2.  **Clone the Repo**:
    ```bash
    git clone https://github.com/Buduthaharishreddy/calculator.git
    cd calculator
    ```
3.  **Run the App**:
    ```bash
    python main.py
    ```

## 🛠️ Project Structure

- `main.py`: Entry point for the application.
- `ui.py`: Handles the Tkinter window and event loop.
- `logic.py`: Contains the `CalculatorLogic` class for mathematical evaluations.
- `test_logic.py`: Simple test script to verify logic functionality.
